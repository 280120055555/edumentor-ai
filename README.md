# 🎓 EduMentor AI — Интеллектуалды Оқу Көмекшісі

> Дипломдық жұмыс | Satbayev University | 6В06102 Computer Science | 2026

## 📱 Демо
http://localhost:3000 (локально)

## 🚀 Технологиялар
| Технология | Мақсаты |
|---|---|
| Next.js 14 | Негізгі фреймворк, PWA |
| TypeScript | Типтік бағдарламалау |
| Firebase | Auth + Firestore |
| Groq API (Llama 3.1) | AI-ментор |
| Web Speech API | Дауыстық интерфейс |
| Tailwind CSS + shadcn/ui | UI/UX |

## ✨ Мүмкіндіктер
- 🤖 AI-ментор (Llama 3.1) — қаз/рус/ағыл тілдерінде
- 🎤 Дауыстық режим — Web Speech API (тегін)
- 📊 Үлгерім аналитикасы — Recharts графиктері
- 📚 AI-импорт — РУП мәтінін автоматты талдау
- 🌐 3 тіл — қазақша, орысша, ағылшынша
- 📱 PWA — смартфонға орнатпай-ақ
